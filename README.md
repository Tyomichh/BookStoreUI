# BookStoreUI

Static single-page UI prototype for an online bookstore website.
This project focuses on layout, styling, and UI structure.

## About the project

This project is a front-end UI demonstration only.
It does not include any backend logic, real authentication, or payment processing.

All forms and interactive elements are visual previews created for learning and practice purposes.

## Features

- Responsive landing page with side navigation menu
- Catalog section with genre filters and “Show more” functionality (UI only)
- Slider section for featured books
- Cart functionality (add/remove items, total price calculation)
- Payment preview form with demo fields (UI only)

## Payment preview notice

The payment preview popup representss a UI preview only.

- No real payment data is collected
- No card transactions are performed
- All fields are demo placeholders for visual representation

## Tech stack

- HTML5
- CSS3 / SCSS
- JavaScript (Vanilla)
- Slick slider
- Vanilla-tilt
- Icomoon
